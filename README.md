## ‚ú® Features

- üèÅ F1-Inspired Design: An interface where a Formula 1 car will be racing while you're in an active session. Each tire compound represents different session durations
- ‚è±Ô∏è Classic Pomodoro: 25-minute work sessions with 5-minute pit stops (configurable)
- üéØ Customizable Intervals: Adjust work and break durations to your pace
- üîä Sound Effects: Team radio notifications when your time is about to end
- üé® Team Themes: Choose your favorite F1 team livery (Ferrari, Mercedes, Red Bull, McLaren, etc.). This changes the entire app's color palette
- üåô Dark/Light Mode: Race day and night mode for comfortable viewing
- üì± Fully Responsive: Works flawlessly on desktop, tablet, and mobile
- üíæ Session Persistence: Never lose your progress with auto-save

## üöÄ Quick Start

### Prerequisites

- Node.js >= 20.0.0
- npm, yarn, pnpm, or bun (preferable)

### Installation
```bash
# Step 1 - Install all the dependencies
bun install

# Step 2 - Run de application
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.
